#  n8n-Workflows

## 🚀 Portolio de Automatización e Integración con IA

Repositorio que contiene una colección de workflows desarrollados en n8n orientados a la automatización de procesos, integración con APIs REST, scraping web y orquestación de agentes de Inteligencia Artificial.

El objetivo de este repositorio es demostrar experiencia en:
- Diseño de pipelines automatizados
- Integración de múltiples servicios
- Orquestación de agentes IA
- Transformación y estructuración de datos
- Buenas prácticas en flujos de trabajo reproducibles

---

## 🧠 Enfoque Técnico

Los workflows incluidos no son ejemplos aislados, sino soluciones estructuradas que aplican principios de:
- Arquitectura modular
- Separación de responsabilidades
- Manejo de múltiples entradas y outputs
- Transformación avanzada de datos
- Procesamiento estructurado de texto
- Automatización basada en eventos
 
---

## 📂 Estructura del repositorio

Cada carpeta dentro de este repositorio representa un proyecto independiente con su documentación correspondiente en su interior. Cada proyecto dentro del repositorio incluye:
- Archivo .json exportado desde n8n
- Documentación específica del proyecto (README.md)
- Diagrama BPMN 2.0 del proceso de negocio

## ⚙️ Cómo ejecutar los workflows
1. Clonar el repositorio
2. Importar el archivo .json en una instancia de n8n
3. Configurar credenciales necesarias (API keys, endpoints, etc.)
4. Ejecutar el workflow desde la interfaz de n8n
